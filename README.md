# Sign Up Form

A Signup form made with HTML/CSS/JS.

## Link: 
-  https://dhote9926.github.io/signup/

## To run locally 

Make sure `node` & `npm` or `yarn` is installed on your system.

```
    > git clone https://github.com/dhote9926/signup.git
    > cd signup/
    > yarn install || npm install
    > yarn run || npm run
```
